# NestJS boilerplate

## Create from scratch
```bash
$ npm i -g @nestjs/cli
$ nest new project-name
```

## Add a new controller
```bash
$ nest g controller controler-name
```



## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
